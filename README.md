
# TerraSeg — LULC Framework



**TerraSeg** is a high-performance, aesthetically pleasing Land Use Land Cover (LULC) satellite segmentation platform. It features a custom sci-fi-themed 3D UI integrated directly into a Python Streamlit application, leveraging Deep Learning for accurate segmentation.



## 🛰️ Overview



The TerraSeg Framework allows users to upload satellite imagery and instantly receive segmentation masks generated by a trained Convolutional Neural Network (CNN). The results and class distributions are displayed in a state-of-the-art interactive dashboard.



### Features

- **Deep Learning Segmentation**: Uses a trained Keras/TensorFlow model to segment satellite images into 6 distinct classes:

  - 💧 Water

  - 🟤 Land

  - 🛣️ Road

  - 🏢 Building

  - 🌿 Vegetation

  - ⚪ Unlabeled

- **Custom Sci-Fi Dashboard**: A highly customized Streamlit frontend injected with interactive HTML/CSS to provide a visually stunning, immersive user experience ("radar-scan effects" and dynamic charts).

- **Instant Analytics**: Real-time breakdown of class distribution and inference metrics.



## 📊 Dataset



This project utilizes the Dubai Satellite Imagery Dataset for semantic segmentation. 



You can find and download the dataset here: 

**[Semantic Segmentation of Aerial Imagery (Dubai Dataset) on Kaggle](https://www.kaggle.com/datasets/humansintheloop/semantic-segmentation-of-aerial-imagery)**



## 🛠️ Technology Stack



- **Frontend Interface**: HTML5, Vanilla CSS (Sci-fi Theme), Streamlit Components

- **Backend Framework**: Python, Streamlit

- **Deep Learning**: TensorFlow, Keras

- **Image Processing**: OpenCV, Pillow (PIL), NumPy



## 🚀 Getting Started



### Prerequisites



You need Python 3.8+ installed. The main dependencies are located in `requirement.txt`.



Required libraries include:

- `streamlit`

- `tensorflow` (`keras`)

- `numpy`

- `opencv-python` (`cv2`)

- `Pillow`



### Installation



1. **Clone the repository** (or navigate to the workspace directory):

   ```bash

   cd path/to/LULC

   ```



2. **Install the required packages**:

   ```bash

   pip install streamlit tensorflow numpy opencv-python Pillow

   ```



3. **Ensure Model Files exist**:

   Make sure `satellite_segmentation_full.keras` is placed in the root directory.



### Running the Application



To launch the TerraSeg dashboard, run the following command in your terminal:



```bash

streamlit run app.py

```



This will start a local server and automatically open the application in your default web browser.



## 📂 Project Structure



- `app.py`: The main Streamlit backend application.

- `index.html`: The custom frontend UI template dynamically injected with data.

- `satellite_segmentation_full.keras`: The trained Keras segmentation model.

- `land.ipynb` & `draw.ipynb`: Jupyter Notebooks documenting the exploratory data analysis, training pipeline, and experimentation.
